# ConverterBot

ConverterBot is a Telegram bot that converts currencies (USD, EUR, RUB) using exchange rates obtained from the Open Exchange Rates API. With ConverterBot, users can easily enter currency and get the conversion of the entered amount.

## Usage

To use ConverterBot, you can add it to your Telegram chat and follow these simple steps:

1. Start a chat with ConverterBot by selecting the "Start" command.
2. Select the button with the currency code you want to convert. For example, "euro".
3. Enter the amount you want to convert.
4. Then the bot will display the conversion amount.

## Technologies used

- Python 3.8.5
- Telegram Bot API
- Open exchange rate API

## Installation

To install ConverterBot, follow these steps:

1. Clone the repository on your local machine.
2. Install the required packages by running: pip3 install -r
3. Add your Telegram bot token and Open Exchange Rates API key.
4. Run the bot.
